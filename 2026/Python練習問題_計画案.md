## Week 4　Python 環境と基本的な数値計算

**到達目標**: Dev Container 内で Python が動くことを確認し、物理定数を使った数値計算ができる。浮動小数点の扱いを意識できる。

### 練習問題

1. **静止エネルギー**  
   光速 c = 3.0×10⁸ m/s、電子の質量 m = 9.109×10⁻³¹ kg として、静止エネルギー E = mc² を J と eV（1 eV = 1.602×10⁻¹⁹ J）の両方の単位で表示せよ。

2. **重力加速度と第一宇宙速度**  
   万有引力定数 G = 6.674×10⁻¹¹ m³/(kg·s²)、地球質量 M = 5.972×10²⁴ kg、地球半径 R = 6.371×10⁶ m として、地表での重力加速度 g = GM/R² と第一宇宙速度 v₁ = √(gR) を計算せよ。`math.sqrt` を使うこと。

3. **ケプラーの第三法則**  
   ケプラーの第三法則 T² ∝ a³（地球基準: a = 1 AU, T = 1 年）を使って、火星（1.524 AU）、木星（5.203 AU）、土星（9.537 AU）の公転周期を年単位で計算して表示せよ。

4. **浮動小数点誤差の確認**  
   `0.1 + 0.2 == 0.3` の結果を確かめよ。次に `abs(0.1 + 0.2 - 0.3) < 1e-10` で判定するとどうなるか。`print(f'{0.1 + 0.2:.20f}')` で内部表現を確認せよ。物理計算で `==` による比較を避けるべき理由を自分の言葉で説明せよ。

## Week 5　NumPy 配列とベクトル演算

**到達目標**: `np.linspace`・`np.array`・ブロードキャストを使い、ループなしに配列全体へ演算できる。for ループとの速度差を体感する。

### 練習問題

1. **単振動（速度比較）**  
   t = 0 〜 4π [s] を 1000 点生成し、x(t) = 2 sin(t) を計算せよ。  
   (a) for ループ版と (b) NumPy 版の 2 通りで書き、`%timeit` で実行時間を比較せよ。  
   配列の要素数を 10⁴、10⁵、10⁶ に増やしたときの速度比も調べよ。

2. **斜方投射**  
   初速度 v₀ = 20 m/s、仰角 θ = 45° として、t = 0 〜 飛行時間 T まで 200 点の配列を作り、x(t) と y(t) を NumPy で一括計算せよ。`np.max()` と `np.argmax()` で最大高度と飛距離を求め、解析解と比較せよ。

3. **プランク黒体放射**  
   波長 λ = 100 nm 〜 3000 nm（2000 点）に対してプランク分布  
   B(λ, T) = (2hc²/λ⁵) / (exp(hc/λkT) − 1)  
   を T = 3000 K、6000 K、15000 K の 3 温度について計算せよ。  
   `np.argmax()` でピーク波長を求め、ウィーンの変位則 λ_max T = 2.898×10⁶ nm·K と比較せよ。  
   ※ h = 6.626×10⁻³⁴ J·s、k = 1.381×10⁻²³ J/K、c = 3.0×10⁸ m/s

## Week 10a　常微分方程式の数値解法

**到達目標**: `scipy.integrate.solve_ivp` で ODE を解き、物理現象をシミュレートできる。エネルギー・角運動量保存の確認ができる。

### 練習問題

1. **単振り子（非線形効果）**  
   θ'' + (g/L) sin θ = 0 を `solve_ivp`（RK45）で解け（L = 1 m, g = 9.81 m/s²）。  
   θ₀ = 5°, 30°, 60°, 90° の 4 条件について t = 0 〜 10 s を計算し、  
   θ(t) を重ねてプロットせよ。各条件の周期を数値的に求め、  
   小振幅近似 T = 2π√(L/g) からのずれを % で示せ。

2. **ローレンツアトラクター**  
   dx/dt = σ(y − x)、dy/dt = x(ρ − z) − y、dz/dt = xy − βz  
   （σ = 10, ρ = 28, β = 8/3）を t = 0 〜 50 で解け。  
   x-z 断面と 3D 軌跡を描け。初期値 x₀ を 10⁻⁸ だけ変えた 2 軌道を重ねて描き、  
   差 |Δr(t)| を半対数プロットして、カオス的発散（リャプノフ指数）を確認せよ。

3. **2体問題（惑星軌道）**  
   太陽を原点に固定し、地球の運動方程式  
   d²r/dt² = −(GM/|r|³) r  
   を solve_ivp で解け。初期条件: x₀ = 1 AU, y₀ = 0, vx₀ = 0, vy₀ = 2π AU/年。  
   軌道を描き、(a) 角運動量 L = r × v と (b) 全エネルギー E = v²/2 − GM/r が  
   時間を通じて保存されているかをプロットして確認せよ。

4. **減衰振動と強制振動**  
   x'' + 2γx' + ω₀²x = F₀ cos(ωt) を solve_ivp で解け。  
   γ = 0.1 [s⁻¹]、ω₀ = 1 [rad/s]、F₀ = 1 として、  
   駆動周波数 ω を 0.5 〜 1.5 [rad/s] でスキャンして最大振幅を求め、  
   共鳴曲線（ω vs 振幅）をプロットせよ。

## Week 10b　フィッティング・コマンドライン・バッチ処理

**到達目標**: 実験データにモデルをフィットできる。コマンドライン引数でパラメータを渡せる。シェルスクリプトでパラメータスキャンを自動化できる。

### 練習問題

1. **スペクトル線のフィッティング**  
   模擬スペクトルデータ（ガウシアン + 定数バックグラウンド）  
   y = A exp(−(x − μ)² / (2σ²)) + b  
   に `scipy.optimize.curve_fit` でフィットせよ。  
   フィット結果の不確かさ（共分散行列の対角成分の平方根）を表示せよ。  
   初期値を大きく外した場合にフィットが失敗することを確かめよ。  
   （天文観測でのスペクトル解析の基本操作）

2. **コマンドライン引数付きスクリプト**  
   `argparse` を使って以下のように実行できる `planck.py` を書け：  
   ```
   python planck.py --temp 6000 --lam_min 100 --lam_max 3000 --output sun.png
   ```  
   引数を変えて T = 3000, 6000, 15000 K の PNG を生成できることを確認せよ。

3. **bash によるパラメータスキャン**  
   上の `planck.py` を使って T = 3000 〜 30000 K（1000 K 刻み）の  
   全 PNG を生成する bash スクリプト `scan_temp.sh` を書け：  
   ```bash
   for T in $(seq 3000 1000 30000); do
       python planck.py --temp $T --output "planck_${T}K.png"
   done
   ```  
   （スパコンのジョブスクリプトの典型的なパターン）

4. **単振り子の周期スキャン**  
   θ₀ = 5°〜 85°（5° 刻み）の 17 条件について、  
   Week 10a の数値解から周期を自動検出し（ゼロ交差を `np.where` で探す）、  
   解析的な小振幅近似との比較グラフを作れ。  
   結果を `pendulum_periods.csv` に保存せよ。
