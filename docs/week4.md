#### 自分の作業ファイル `week4.py` を作る

`docs` の更新が確認できたら、**授業用フォルダ（自分の作業フォルダ）の下** に `week4.py` という新しいファイルを作成する。VS Code のエクスプローラーで授業用フォルダを右クリック → 「新しいファイル」→ `week4.py` と入力すればよい。

今週からは、解説中の **グレイ背景の Python コード** や 例１，２，… の例題部分を、この `week4.py` に **`# %%` 記法** を使って書き写しながら動作を試していく。`# %%` 記法とは、Python ファイルの中をセル単位に区切って実行できる VS Code の機能で、Jupyter Notebook のような対話的な実行ができる。書き方は次のとおり：

```python
# %%
# セル1：変数と型を試す
x = 10
y = 3.14
print(x, y)

# %%
# セル2：算術演算を試す
print(x + y)
print(x ** 2)
```

`# %%` の行が **セルの区切り** になる。各セルの上に表示される `Run Cell` をクリックする（または `Shift + Enter`）と、そのセルだけが実行され、結果がインタラクティブウィンドウに表示される。これにより、コード全体を最初から実行し直さずに、解説中のコード片を一つずつ試して動作を確認できる。

> **指示**: 以降に出てくるグレイ背景のコードは、**そのまま読むだけでなく、必ず `week4.py` に `# %%` で区切りながら入力して動作を確認すること**。例題部分（例１，例２，…）も同様に、自分で入力して試すこと。コピペではなく **自分の手で打ち込む** ことで身につく。

### 文法解説

#### 変数と型
```python
x = 10          # int（整数）
y = 3.14        # float（小数）
z = 3 + 2j      # complex（複素数）
name = "wada"   # str（文字列）
```

#### 算術演算子
| 演算子 | 意味 | 例 | 結果 |
|--------|------|----|------|
| `+` `-` `*` | 加減乗 | `3 + 2` | `5` |
| `/` | 除算（float） | `7 / 2` | `3.5` |
| `//` | 整数除算 | `7 // 2` | `3` |
| `%` | 余り | `7 % 2` | `1` |
| `**` | 冪乗 | `2 ** 10` | `1024` |

#### print と f-string
```python
v = 11200.0   # m/s
print(f"脱出速度: {v:.2f} m/s")        # 小数点以下 2 桁
print(f"脱出速度: {v:.3e} m/s")        # 指数表記
print(f"脱出速度: {v/1000:.1f} km/s")  # 単位変換して表示
```

#### リスト・タプル・for ループ（この先ずっと使う基本道具）

##### リスト（list）― データをまとめて入れる箱
```python
masses = [9.109e-31, 1.673e-27, 3.344e-27]   # 電子・陽子・重陽子の質量
print(masses[0])    # 最初の要素（番号は 0 から始まる！） → 9.109e-31
print(masses[2])    # 3 番目 → 3.344e-27
print(len(masses))  # 要素数 → 3
```

##### タプル（tuple）― 変更しないデータのまとまり
```python
earth = ("地球", 1.000)   # 名前と軌道長半径をセットにする
print(earth[0])           # → "地球"
print(earth[1])           # → 1.000
```
リストは `[ ]`、タプルは `( )` で作る。タプルは後から中身を変更できない（定数的なデータに便利）。

##### for ループ ― 繰り返し処理
```python
# リストの要素を 1 つずつ取り出して処理
planets = ["水星", "金星", "地球"]
for name in planets:
    print(name)           # → 水星、金星、地球 が順に表示される

# range で回数を指定して繰り返す
for i in range(5):        # i = 0, 1, 2, 3, 4
    print(i ** 2)         # → 0, 1, 4, 9, 16

# タプルのリストを for で展開する（例 2 で使う）
data = [("水星", 0.387), ("金星", 0.723), ("地球", 1.000)]
for name, a in data:      # タプルの中身を name と a に分けて受け取る
    print(f"{name}: {a} AU")
```

> **ポイント**: `for 変数 in リスト:` の後の行は **インデント（半角スペース 4 つ）** が必要。  
> インデントがないとエラーになる。VS Code では Enter を押すと自動でインデントされる。

#### `import` ― 他の人が作った道具を借りてくる

Python には最初から使える機能（`print`, `len`, `int` など）のほかに、
**モジュール（module）**と呼ばれる「機能をまとめたファイル」がたくさん用意されている。
`import` はそのモジュールを読み込んで使えるようにする命令。

```python
import math          # math モジュールを読み込む
print(math.pi)       # math の中にある pi を使う → 3.141592653589793
```

`import` には主に 3 つの書き方がある。

```python
# ① モジュール全体を読み込む（基本形）
import math
math.sqrt(2)         # "math." をつけて使う

# ② 短い別名をつけて読み込む（長い名前を省略したいとき）
import numpy as np
np.array([1, 2, 3])  # "np." で使える

# ③ モジュールの中から特定の関数だけ読み込む
from math import sqrt, pi
sqrt(2)               # "math." なしで直接使える
```

| 書き方 | 使い方 | よく見る例 |
|--------|--------|-----------|
| `import X` | `X.関数名()` | `import math` |
| `import X as 別名` | `別名.関数名()` | `import numpy as np` |
| `from X import 関数名` | `関数名()` | `from math import sqrt` |

> **この授業でよく使うモジュール:**  
> `math`（数学関数）、`numpy`（配列・数値計算）、`matplotlib`（グラフ描画）、`scipy`（科学計算）  
> いずれも Dev Container に最初からインストール済みなので、`import` するだけで使える。

#### math モジュール
```python
import math
math.pi          # 円周率 π
math.e           # 自然対数の底 e
math.sqrt(2)     # √2
math.sin(math.pi / 6)   # sin(30°) → 0.5
math.log(math.e)         # 自然対数 → 1.0
math.log10(1000)         # 常用対数 → 3.0
```

### 例 1 ― 運動エネルギーと静止エネルギー

```python
# week4_ex1.py
import math

# 物理定数
c = 3.0e8           # 光速 [m/s]
eV = 1.602e-19      # 1 eV [J]

# 電子
m_e = 9.109e-31     # kg
E_rest = m_e * c**2
print(f"電子の静止エネルギー:")
print(f"  {E_rest:.4e} J")
print(f"  {E_rest / eV / 1e6:.4f} MeV")

# 陽子
m_p = 1.673e-27     # kg
E_proton = m_p * c**2
print(f"\n陽子の静止エネルギー:")
print(f"  {E_proton:.4e} J")
print(f"  {E_proton / eV / 1e6:.4f} MeV")
```

**実行結果の確認ポイント**: 電子 ≈ 0.511 MeV、陽子 ≈ 938.3 MeV になっていれば正解。

### 例 2 ― ケプラーの第三法則

```python
# week4_ex2.py
import math

# ケプラーの第三法則: T = (a^(3/2)) 年  （a は AU 単位）
planets = [
    ("水星", 0.387),
    ("金星", 0.723),
    ("地球", 1.000),
    ("火星", 1.524),
    ("木星", 5.203),
    ("土星", 9.537),
]

print(f"{'惑星':8s}  {'軌道長半径 [AU]':16s}  {'公転周期 [年]':14s}")
print("-" * 46)
for name, a in planets:
    T = a ** 1.5
    print(f"{name:8s}  {a:16.3f}  {T:14.3f}")
```

**実行結果の確認ポイント**: 地球 = 1.000 年、木星 ≈ 11.9 年になっていれば正解。
